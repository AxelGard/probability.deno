# probability.deno
probability.js deno / JSR port 


[JSR package](https://jsr.io/@axelgard/probability)

## Install 

```bash
deno add jsr:@axelgard/probability
```

## Example

```typescript
import {probability} from "@axelgard/probability";

function main(): void {
    if (probability(0.5)){
        console.log("50/50 chance of True")
    }
} 


```
